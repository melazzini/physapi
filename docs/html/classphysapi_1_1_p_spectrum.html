Detailed Description

template<phys_size energyIntervals>
class physapi::PSpectrum< energyIntervals >

This class represents a raw spectrum(or photon distribution).

Template Parameters
energyIntervalsnumber of energy intervals.

A raw spectrum is simply a vector of energy-value pairs where the energy corresponds to the energy interval and the value is the number of photons in that interval.

Example

int main()
{
PSpectrum<10> spectrum{3, 20, eSpectrumScale::LIN};
spectrum.addPhoton(4);
spectrum.addPhoton(5);
spectrum.addPhoton(9.9);
spectrum.addPhoton(10);
spectrum.addPhoton(7.8);
cout << spectrum << endl;
return 0;
}

output:

1 5.55 1
