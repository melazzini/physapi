Detailed Description

This class represents a vector in the XY plane.

You can use two different coordinate systems:

  • Rectangular(Cartesian): \(x,\; y\).
  • Polar : \( \rho,\; \theta\).

In addition, you can perform all common vector operations( \( +, - \) , etc), manipulate them and perform with them other operations like printing their components with std::cout, etc.
