Construct a new radius vector with the specified coordinates. 

As you can see, if you need to create a 3D-\/vector in rectangular coordinates, then x1 represents x and x2 represents y, x3 represents z. On the other hand, if you need a polar vector, then x1 represents $ \rho $ x2 represents $ \theta $, and x3 represents $ \phi $.
